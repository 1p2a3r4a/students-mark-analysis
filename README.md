# 📊 Student Marks Analysis

## 📌 Project Overview
This project analyzes student performance using Python.

## 🎯 Objectives
- Analyze student marks
- Study impact of study hours
- Compare performance

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

## 📂 Dataset
Contains:
- Name
- Gender
- Study Hours
- Marks (Math, Science, English)

## 🔍 Steps
1. Load dataset
2. Clean data
3. Calculate average
4. Analyze data
5. Create graphs

## 📊 Insights
- More study hours → higher marks
- Topper identified
- Subject comparison done

## ▶️ How to Run
pip install pandas matplotlib  
python studentsanalysis.py
